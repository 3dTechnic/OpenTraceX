# Software Guide
To use the software, just download the flaskServer.py, trace_to_svg.py and requirements.txt.

To set it up for the first time, put all of those files into one folder.
## Python is required, so if you do not have Python, install it now.
- Navigate to that folder in the command line.
- Run the following command: **pip install -r requirements.txt".**
- If this doesn't work, switch the **pip** for **pip3**

### Now, you're almost ready to go.
The project is ready to be turned on, you can turn it on by running **python flask_server.py** or **python3 flask_server.py** (whichever of these works without errors).
